自建Bitwarden

全部网站一个密码,显然不安全,而每个网站一个密码对人的难度可想而知,好麻烦啊,而且很难清晰地记忆,而密码保存在其他的软件厂商的地方,厂商的声誉和以前被暴库也会让人忧心忡忡,因此数据在自己手中,相对来说也就保证了自己的安全,因此自建bitwarden是一种不错的选择.

分析

优点分析

  1. 开源,可审计,保证数据的安全和算法的安全
  2. 自建安全性有保证
  3. 有很强大的软件系统和插件系统,保证了软件的普遍适用性

综上,此软件在安全性上有保证且使用起来比较方便,因此采用此种方案,比较安全

宝塔面板搭建流程

众所周知,宝塔面板可以减轻开发者搭建开发环境的痛苦和折磨,一键就可完成环境的搭建并且拥有一个完整的面板系统,因此,推荐宝塔面板或许会帮助开发者更快的配置环境

优点

  1. 快速配置环境

缺点

  1. 直到当下:2020 -4-1 宝塔面板仍然有很多未知的错误,如果不是你的问题,很有可能是宝塔的问题.
  2. 直至当下:2020-4-1 宝塔面板被强攻的次数有很多,推荐换端口,换账号密码

宝塔面板可以做什么

  1. 快速搭建网站(直接是上传网页到服务器和绑定域名以及宝塔提供一件安装脚本的网站程序(WordPress等))
  2. 快速开启新的独立数据库
  3. 快速开启FTP客户端
  4. 较为容易的监控日志
  5. 较为简单的管理防火墙
  6. 较为简单的开启计划任务

所需资源

  1. 服务器一台
  2. 可以联网的计算机一个
  3. shell软件一个(可有可无)
  4. 所需资源 bt.cn

Springboot后端的JPA实现

springboot后端开发有多种类,JPA实现可以少写很多的mapper文件,当然使用生成器也可以生成相应的文件,用生成器生成的情况下次再写,此次记录一下使用JPA的形势实现一下后端的项目

所需资源

  1. 代码编辑器idea

  2. 计算机一台

  3. 带有数据库的服务器(电脑)一台

  4. 数据库连接软件 推荐Navicat

优点

明显加快开发速度,不用写mapper和mapper对应的dao减少了很多常规化的代码量(经常使用的),相对于其他的方式,显著加快了CRUD的速度.
